# Contributing

Thanks for helping improve Compaction State Guard.

## Development Setup

```bash
npm install
npm test
```

## Pull Requests

- Keep changes focused and explain which lifecycle event or recovery behavior is affected.
- Add or update tests for state capture, reset handling, TTL behavior, and prompt injection behavior.
- Avoid committing credentials, local user IDs, absolute local paths, or machine-specific configuration.
- Keep documentation public-facing and avoid embedding private operational policies.

## Security

Please do not open public issues containing secrets, tokens, private session transcripts, or sensitive logs. Report sensitive security findings privately to the repository owner.
